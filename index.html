Vssue

A Vue-powered Issue-based Comment Plugin

Get Started →

Comments

Enable comments support for static pages, based on the issue system of code hosting services.

Multi-platform

Support multiple code hosting services, including GitHub, GitLab, Bitbucket and Gitee.

Use Vue

Use Vssue as a Vue plugin, and can be integrated in your Vue App easily.

# Demo

Demo of current page is for GitHub.

Go to the Demo Page for all supported platforms.

After you comment on an issue, the platform will send you emails when new comments come. The emails might be annoying when new users come and leave comments on this demo page.

You can go to the issue page and unsubscribe the notifications.
